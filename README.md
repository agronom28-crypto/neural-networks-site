# 🧠 NeuralHub — Сайт популярных нейросетей

Сайт-каталог популярных нейронных сетей с системой авторизации пользователей.

## Страницы

- `index.html` — Главная страница
- `networks.html` — Каталог нейросетей с фильтрацией
- `login.html` — Вход в аккаунт
- `register.html` — Регистрация

## Функционал

- ✅ Регистрация и авторизация (localStorage)
- ✅ 18 популярных нейросетей в каталоге
- ✅ Фильтрация по категориям (LLM, изображения, аудио, видео, код)
- ✅ Адаптивный дизайн (мобильный + десктоп)
- ✅ Тёмная тема

## Категории нейросетей

| Категория | Примеры |
|---|---|
| Языковые (LLM) | GPT-4o, Claude, Gemini, LLaMA |
| Изображения | DALL-E 3, Midjourney, Stable Diffusion, Flux |
| Аудио | Whisper, ElevenLabs, MusicGen |
| Видео | Sora, Runway, Kling AI |
| Код | GitHub Copilot, Cursor, DeepSeek Coder |

## Запуск

Открой `index.html` в браузере — всё работает без сервера (чистый HTML/CSS/JS).
